# Introduction
This is my Degree final year project - a Meta Assembler in Modula-2.

# Abstract
The project documents the historical development of the meta-assembler, the possible types available, principles
of operation and methods of construction. Various implementations and suggested methods of implementation are documented.

A sample implementation of a meta-assembler is developed using the principles of Object Oriented Software
Construction in the high level language Modula–2. The assembler is written so as to conform to the IEEE Proposed
Assembly Language Draft Standard of 1979.

The report documents the complete development process from
Problem Definition to Implementation and Testing. It is concluded that the implementation developed compares
well with previous meta-assemblers in terms of speed, flexibility and the ease with which assembler definitions
can be written.
